# CNNIC第47次中国互联网络发展统计报告观后感

### 互联网基础资源
   中国近几年IP地址和域名逐年上升，国际出口宽带的增长率总体呈现上升趋
势，表明中国互联网近几年发展势头良好。

### 互联网资源应用
   统计报告结果表明，移动互联网接入流量和网页数量都在上升，网站以及APP
的数量正在减少。说明中国使用互联网的人数和时长增加，但市场出现饱和现象，
需要不断创新出更先进的技术支持互联网的发展。

### 互联网接入环境
   根据统计报告，互联网中使用移动手机接入的人数最多。在现在这个迅速发展
的快时代，移动手机携带便捷，最符合人们的生活方式。人均上网方式正在减少，
说明人们不在一直盯着屏幕，更加注重生活，创建更加健康的生活方式。

### 网民规模和属性
   网民规模和互联网普及率不断增加，表明中国科技迅速发展，互联网覆盖率增高
和智能通讯工具普及广，正逐渐往数字化转型发展。但也有一些非网民，主要原因是
年龄大不懂网络，由于没有互联网，导致非网民的生活有些不便的地方，例如疫情期
间没有健康码，出行不便，不能移动支付等等...我们应该提升网络服务便利化和智
能化，让不懂互联网的人生活不受影响。
   网民属性广泛，各个年龄段，各行各业都有，说明互联网已经深入人们的生活。
   我国还利用互联网进行网络扶贫，主要手段是宣传各贫困地区的特色农产品，这一
行为为我国脱贫攻坚战提供了有力保障，还为贫困地区提供了良好的教育资源，使贫困
地区的孩子们受到更好的教育。

### 互联网发展状况
   1.商务交易
   近几年使用互联网商品交易的使用率在逐步提高，电商形式不断发展，激活了城乡
消费循坏，稳定了外贸，为建设以国内循环为主，国内国际双循坏的新发展格局提供了
保障，促进我国经济迅速发展。
   2.公共服务
   随着外卖行业的发展，服务业模式创新，带动了一系列产业的发展。还有网络支付等
共同助力中国朝数字化转型。
   在在线医疗方面，互联网为其构建了完整的医疗服务生态，盘活了医疗资源，让更多
人享受先进的医疗技术，保障了人们的身体健康。
   在政务方面，各地政府实行“一网通办”并不断完善，使人民参与政治更加便利，提高了
公民的参政热情，更好建设高效政府。

### 安全状况
   目前随着互联网的迅速发展，网络安全问题日益严重。安全状况较差，主要问题是手机
诈骗增加，需要加强这方面的建设，为网民提高更良好的上网环境。网民也应该提高安全意
识，提防诈骗。
